# Shelby Node Commands

## Node Control
- Start node: `./shelby start`
- Stop node: `./shelby stop`
- Restart node: `./shelby restart`
- Check status: `./shelby status`

## Logs
- View logs: `./shelby logs`
- Follow logs live: `./shelby logs -f`

## Maintenance
- Update node software: `./shelby update`
- Backup node data: `./shelby backup <path>`
